Starting of 3D_Portfolio Web Design with Tailwindcss , Three js
1.Create a folder "Portfolio"
2.Open Terminal in vscode
3.Type this in Terminal // npm create vite@latest ./ -- --template react //
4.Installing Tailwind // npm install -D tailwindcss //
5.Initialize Tailwind // npx tailwindcss init //
6.React Tilt package installation // npm install --legacy-peer-deps @react-three/fiber @react-three/drei maath react-tilt react-vertical-timeline-component @emailjs/browser framer-motion react-router-dom
7.Tailwind work with vite We need to install this // npm install --legacy-peer-deps -D postcss autoprefixer //
8.Initialize tailwindcss // npx tailwindcss init -p //
